# Library Management System

A complete Library Management System project with source code built in C# Windows Forms, using text files as a database.

## Features

- Book Management (Add, Update, Delete, Search)
- Student Management
- Book Issuing System
- Book Return System
- User-friendly Windows Forms Interface
- Text file-based database system

## Technology Stack

- **Language**: C#
- **Framework**: .NET Framework 4.5.2/4.8
- **UI**: Windows Forms
- **Database**: Text Files
- **IDE**: Visual Studio
